# 🔥 消防灭火策略游戏

一个结合物理喷射和策略布局的消防灭火游戏。

## 🎮 游戏特色

- **双阶段玩法**: 准备阶段布局设施 → 战斗阶段喷射灭火
- **物理引擎**: 真实的抛物线水柱轨迹
- **策略元素**: 放置消防栓、防火墙、消防员
- **5个关卡**: 从教学关卡到终极挑战
- **响应式设计**: 支持PC和移动设备

## 🎯 玩法说明

### 游戏流程

1. **准备阶段** (30秒)
   - 点击侧边栏选择设施
   - 在画布上点击放置
   - 合理布局防御设施

2. **战斗阶段**
   - 使用键盘或鼠标控制水枪
   - 调整角度和力度
   - 熄灭所有火焰

3. **结算**
   - 根据表现获得1-3星评价
   - 解锁下一关卡

### 控制方式

**键盘控制**:
- `↑` / `↓` - 调整角度
- `←` / `→` - 调整力度
- `空格` - 发射水柱
- `ESC` - 暂停

**鼠标控制**:
- 移动鼠标 - 调整角度和力度
- 按住左键 - 持续发射

## 🚀 快速开始

### 安装测试依赖

```bash
npm install
```

### 在线运行

```bash
# 克隆项目
git clone <repository-url>
cd fire-fighter-game

# 启动服务器
./start.sh
```

然后在浏览器中访问: http://localhost:8080

## 🎨 技术栈

- **Canvas 2D** - 主渲染引擎

- **粒子系统** - 火焰、水花、烟雾效果
- **物理引擎** - 抛物线轨迹、风力系统

## 💡 游戏技巧

1. **准备阶段**：优先在火源附近放置消防栓
2. **节省水量**：精确瞄准，避免浪费
3. **利用风力**：根据风向调整角度
4. **保护建筑**：优先保护耐火性差的木屋
5. **组合设施**：消防栓+消防员组合效果更佳

## 🔧 开发说明

### 修改配置

编辑 `js/constants.js` 文件可以修改：

- 游戏参数 (画布大小、FPS等)
- 物理常量 (重力、风力)
- 游戏配置 (水量、火焰强度)
- 关卡数据

### 添加关卡

在 `js/constants.js` 的 `LEVEL_DATA` 数组中添加新关卡：

```javascript
{
    name: "关卡名称",
    description: "关卡描述",
    buildings: [
        { type: "WOOD", x: 100, y: 400 },
        // 更多建筑...
    ],
    initialFires: [0],  // 初始着火的建筑索引
    wind: 2,            // 风力
    initialWater: 1000, // 初始水量
    time: 60,           // 时间限制(秒)
    targetScore: 1000   // 目标得分
}
```
